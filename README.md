# Self-Hosted Warehouse App with Desktop Support

"The code is functional, but it needs optimization for better performance and efficiency."

This is a self-hosted warehouse management application built with **Spring** and **JavaFX** for desktop support.

### Why Spring?
Spring is an ideal choice for this application due to its powerful, scalable framework, providing flexibility for both backend and future API expansion.

### Desktop Support (Completed)
The desktop version uses **JavaFX** to provide a rich, user-friendly interface. JavaFX offers pre-built components, which saves time and provides consistency without the need for excessive customization.

### API (Not Completed)
While the backend is ready, the API for integration is still under development.
